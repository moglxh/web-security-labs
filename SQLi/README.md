# SQL Injection Labs

## Overview

Collection of SQL Injection labs completed through PortSwigger Web Security Academy and TryHackMe.

## Skills Demonstrated

- Authentication bypass
- UNION attacks
- Data extraction
- Blind SQL injection
- Time-based SQL injection

## Labs Completed

- Hidden Data
- Login Bypass
- UNION Columns
- UNION Data Extraction
- UNION Multiple Values
- Blind SQLi (Conditional)
- Blind SQLi (Conditional Errors)
- Blind SQLi (Time Delay)

## Tools Used

- Burp Suite
- Browser Dev Tools
- Manual Payload Crafting
