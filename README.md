# Web Security Labs

A collection of practical web application security labs completed through PortSwigger Web Security Academy and personal research.

## Topics

- SQL Injection
- Cross Site Scripting (XSS)
- Server Side Request Forgery (SSRF)
- Authentication Vulnerabilities
- Information Disclosure
- OS Command Injection

## Objectives

- Understand vulnerability root causes
- Develop exploitation methodology
- Document attack paths
- Learn remediation strategies

## Skills Demonstrated

- HTTP Request Analysis
- Burp Suite Usage
- Vulnerability Validation
- Attack Surface Mapping
- Secure Coding Awareness
