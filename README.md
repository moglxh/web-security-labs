# Web Security Labs

A collection of practical web application security labs completed through PortSwigger Web Security Academy, TryHackMe, and personal security research.

## Categories

### SQL Injection

* Authentication bypass
* UNION attacks
* Blind SQL injection
* Time-based attacks

### Cross-Site Scripting (XSS)

* Reflected XSS
* Stored XSS
* Filter bypass techniques

### Server-Side Request Forgery (SSRF)

* Localhost access
* Backend service discovery
* Whitelist and blacklist bypasses

### Cross-Site Request Forgery (CSRF)

* Missing protections
* Token validation weaknesses
* Method bypasses

### Authentication Vulnerabilities

* Authentication logic flaws
* Session handling weaknesses

### Information Disclosure

* Sensitive data exposure
* Debug information leakage

### OS Command Injection

* Command execution vulnerabilities
* Input validation failures

## Skills Demonstrated

* HTTP request analysis
* Burp Suite usage
* Vulnerability validation
* Exploitation methodology
* Attack surface mapping
* Security documentation

## Platforms

* PortSwigger Web Security Academy
* TryHackMe
* Personal Security Labs
